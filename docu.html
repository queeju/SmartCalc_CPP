SmartCalc v2.0 by Greaterm

Description of the project:

This program is a versatile calculator that allows users to perform various calculations, including evaluating arithmetic expressions, plotting graphs of functions, and calculating credits. The program provides an intuitive user interface to enter expressions and view the results.

Arithmetic Expression Evaluation:

  • Users can enter arbitrary bracketed arithmetic expressions in infix notation using common arithmetic operators such as addition, subtraction, multiplication, division, power, and modulus.
  • The calculator accurately evaluates the expressions up to 13 decimal places and can handle expressions up to 255 characters.
  • It supports mathematical functions such as cosine, sine, tangent, arc cosine, arc sine, arc tangent, square root, natural logarithm, and common logarithm.

Graph Plotting:

  • Users can plot graphs of functions given in infix notation with the variable 'x'.
  • The program provides a user-friendly coordinate system with marked scales and an adaptive grid.
  • User can use scroll wheel to zoom in and out and drag the image to move along the grid.
  • The domain and codomain of the function are limited to the range from -1,000,000 to 1,000,000, but the user can specify the displayed domain and codomain for plotting.
  • To create the image of the plot, user must input the equation, click the graph icon ad click "plot" button.

Credit Calculator:

  • The program includes a special mode known as the "credit calculator."
  • Users can input details like the total credit amount, term (duration), interest rate, and type of credit (annuity or differentiated).
  • Based on the input, the calculator provides information such as the monthly payment, overpayment on credit, and total payment.

Deposit Calculator:

  • The deposit profitability calculator is a specialized mode within the SmartCalc program designed to help users analyze the potential returns and outcomes of various deposit scenarios.
  • It enables users to input specific parameters related to their deposit accounts and receive comprehensive information about the accrued interest, and final deposit amount at the end of the term.

The calculator offers great flexibility and accuracy, making it suitable for various mathematical computations and financial calculations. Users can efficiently evaluate complex arithmetic expressions, visualize functions' graphs, and make informed financial decisions using the credit calculator mode. With its user-friendly interface and accurate results, the program caters to a wide range of mathematical and financial needs.
